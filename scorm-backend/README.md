# SCORM File Uploading System – Backend

This repository contains the backend implementation of the SCORM File Uploading System built using Java Spring Boot.

The backend handles SCORM file upload, processing, and communication with the frontend application.

## Technology Stack

Java
Spring Boot
Spring Web
Spring Data JPA
Hibernate
MySQL / H2 Database

## Backend Responsibilities

* Handling SCORM file uploads
* Processing SCORM packages
* Managing course content
* Providing REST APIs for frontend integration
* Managing metadata related to uploaded SCORM content

## API Features

* Upload SCORM package
* Retrieve uploaded SCORM files
* Manage SCORM course data

## Running the Backend

Clone the repository

Import the project into Spring Tool Suite or any Java IDE

Run the application as a Spring Boot Application

The backend server will start on

http://localhost:8080
