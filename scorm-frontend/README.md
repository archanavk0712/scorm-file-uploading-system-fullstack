# SCORM File Uploading System – Frontend

This repository contains the frontend implementation of the SCORM File Uploading System built using modern web technologies.

The frontend allows users to upload SCORM packages and interact with course content through a web interface.

## Technology Stack

Next.js / React
JavaScript
HTML
CSS

## Features

* Upload SCORM packages
* Display uploaded SCORM content
* Communicate with backend APIs
* Manage SCORM course interaction

## API Communication

The frontend communicates with the backend using REST APIs to upload and retrieve SCORM packages.

## Running the Frontend

Clone the repository

Install dependencies

npm install

Run the application

npm run dev

Open the application in the browser

http://localhost:3000
